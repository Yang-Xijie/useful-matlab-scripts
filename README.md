# useful-matlab-scripts
seperate `MATLAB` scripts which are useful in university courses

欢迎大家提PR添加你觉得在大学学习期间很实用的`MATLAB scripts`！

是开源的项目，所以如果有别的项目要直接用这些`useful scripts`的话，毕竟`MIT`协议，放个`LICENSE`链接直接用吧

希望能对大家有帮助！
